Analisador de DXF

Uma aplicação desktop para validação e verificação de desenhos para corte CNC.

Tela de verificação, exibindo status de cada arquivo.

Visualização do contorno da peça analisada.

O Desafio

Arquivos DXF recebidos de diversas fontes frequentemente continham erros como linhas sobrepostas, contornos abertos ou camadas incorretas, causando paradas de máquina e desperdício de material. A verificação manual era lenta e ineficiente.

A Solução

Desenvolvi uma aplicação desktop em Python utilizando a biblioteca Tkinter (ou PyQt/PySide) para a interface gráfica. A ferramenta permite que o usuário selecione uma pasta de arquivos DXF, e o programa automaticamente analisa cada um, verificando uma lista de problemas comuns. O status (OK, Erro, Atenção) é exibido em uma tabela, junto com detalhes do problema encontrado, permitindo uma correção rápida antes do envio para a máquina CNC.

Tecnologias Utilizadas:

  • Python
  • ⚙️ Biblioteca Tkinter
  • ⚙️ Biblioteca Ezdxf
