DBX - Drawing Builder X

App para geracao de desenhos e orçamentos de peças cortadas no Laser e Plasma

Interface Principal com os botoes para iniciar um novo projeto ou editar algum existente

Tela com o projeto já iniciado, as peças no projeto são apresentadas em uma lista

Desenvolvimento de API para Otimização de Matéria-Prima (Nesting 2D) com Python e Rectpack

O Desafio

Nosso principal desafio era a elaboração de um grande volume de desenhos técnicos, um processo manual, repetitivo e propenso a erros.

A Solução

Como resultado direto da automação, alcançamos um ganho de eficiência massivo: o tempo necessário para o ciclo completo de desenho foi reduzido em mais de 60%. Isso permitiu não apenas aumentar a vazão de projetos, mas também realocar nossa equipe para atividades mais estratégicas e de maior valor agregado.

Tecnologias Utilizadas:

  • Python
  • Tkinter (GUI)
  • Rectpack (Biblioteca Python)
  • Pandas (Biblioteca Python)
