Gerador de DXF

Uma ferramenta web para automação de desenhos técnicos.

Tela principal da aplicação.

Detalhe da funcionalidade de furação automática / furação manual.

Opção para atribuir bloco de informações nas peças (Calcula peso com base na espessura).

O Desafio

A criação manual de arquivos DXF para peças simples e repetitivas era um processo demorado e propenso a erros, consumindo horas valiosas de engenheiros e projetistas. O objetivo era criar uma solução que automatizasse essa tarefa.

A Solução

Desenvolvi uma aplicação web com uma interface intuitiva onde o usuário pode inserir parâmetros como dimensões, furos e offsets. O front-end (HTML, CSS, JS) captura esses dados e os envia para um backend em Python (usando Flask/Django) que processa as informações e gera um arquivo .DXF válido, pronto para download e uso em máquinas CNC.

Tecnologias Utilizadas:

  • HTML5
  • CSS3
  • JavaScript
  • Python
  • ezdxf
